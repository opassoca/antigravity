## 🧬 O Coração do Arsenal
O **Ecosystem PDK** é a estrutura unificada que organiza e gerencia os scripts de IA do Arquiteto. Ele utiliza uma arquitetura **Master-Consumer**, onde um único painel de autenticação centraliza as identidades para todos os outros módulos.

## 🚀 Estrutura Modular

### 🔒 [Master] Gemini Auth (`/auth`)
O painel mestre standalone. Gerencia tokens OAuth2, monitora cotas e define a conta ativa em todo o sistema.
*   **Truth Source:** `~/.gemini-auth/google_accounts.json`

### ⚡ [Consumer] AGY3 Proxy (`/agy3`)
Túnel de execução para o motor oficial Gemini CLI. Consome a identidade ativa do mestre e aplica isolamento de ambiente.

### 🔄 [Consumer] GMN Switcher (`/gmnswitcher`)
Interface de troca rápida e gestão de perfis integrada ao mestre.
